# Retail Sales Analysis using SQL

## Project Overview
This project analyzes a retail sales dataset to identify business insights such as profitability, regional performance, customer value, and discount impact.

The goal of this project is to demonstrate strong SQL skills and analytical thinking required for data analyst roles.

---

## Dataset
The dataset contains transactional retail data including:

- Order details
- Customer details
- Product details
- Sales
- Profit
- Discounts
- Regions
